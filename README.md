# 🔥 InsightX – Sales Analytics Dashboard

## 📊 Project Overview

InsightX is an interactive Sales Analytics Dashboard developed using Power BI to provide meaningful business insights into sales performance, profitability, customer behavior, and revenue trends.

This dashboard helps businesses monitor key performance indicators (KPIs), analyze sales trends, identify top-performing products and customers, and support data-driven decision-making.

---

## 🚀 Features

* Interactive KPI Cards
* Revenue & Profit Analysis
* Customer Insights Dashboard
* Product Performance Tracking
* Executive Reporting Page
* Dynamic Date Filtering
* Interactive Page Navigation
* Business Performance Monitoring
